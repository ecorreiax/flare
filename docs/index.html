Flare CLI

Flare is a lightweight CLI for managing databases in Go projects.

Star Follow

  • $ flare database

    Create

    It creates a new schema in the remote database if a previous schema doesn't already exists.

    $ flare database create

    Drop

    It drops the schema from the remote database.

    $ flare database drop

    Seed

    Apply mock's data from your seed's file to the remote database.

    $ flare database seed

  • $ flare generate

    Migration

    A new versioned SQL migration file is created with a default template.

    $ flare generate migration [MODEL] [COLUMNS]
